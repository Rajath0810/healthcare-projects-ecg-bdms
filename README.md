# healthcare-projects-ecg-bdms
ECG Disease Detection using Machine Learning and Blood Donor Management System

## Overview
This repository includes complete project documentation, system design, and implementation details.

This repository contains two academic projects focused on healthcare applications:

1. ECG-based Cardiovascular Disease Detection using Machine Learning  
2. Blood Donor Management System (Web Application)

These projects demonstrate practical implementation of machine learning and full-stack development to solve real-world healthcare problems.



## Project 1: ECG Disease Detection using Machine Learning

### Problem Statement
Traditional ECG analysis relies on manual interpretation, which is time-consuming and prone to human error. There is a need for an automated system to improve accuracy and efficiency.

### Proposed Solution
This project presents a machine learning-based system that processes ECG images, extracts relevant features, and classifies them into different categories of cardiovascular conditions.

### Methodology
- Input ECG images
- Preprocessing (cropping, resizing, noise removal)
- Feature extraction using Principal Component Analysis (PCA)
- Classification using k-Nearest Neighbors (KNN)
- Output prediction of disease category

### Technologies Used
- Python
- Machine Learning
- Image Processing
- Anaconda / Python IDE

### Features
- Automated disease detection
- Improved diagnostic accuracy
- Non-invasive analysis
- Scalable for larger datasets

### Limitations
- Requires high-quality ECG images
- Performance may degrade with noisy data
- Dependent on training dataset quality


## Project 2: Blood Donor Management System

### Problem Statement
Manual blood donor management systems are inefficient, error-prone, and lack proper coordination between donors, hospitals, and blood banks.

### Proposed Solution
A web-based system designed to manage donor information, blood inventory, and donation campaigns efficiently.

### System Functionality
- Donor registration and login
- Blood group search
- Blood inventory tracking
- Campaign management
- Admin control panel
- Announcement management

### Technologies Used
- HTML, CSS, JavaScript, Bootstrap
- Node.js / PHP
- MySQL Database

### Database Design
- Normalization up to 3NF/BCNF
- ER diagram and schema design included
- Key tables: Donor, Admin, Blood, Campaign, Users

### Features
- User-friendly interface
- Secure authentication
- CRUD operations for data management
- Efficient database handling
- Scalable design

### Challenges and Solutions
- Ensuring data consistency and integrity  
  → Addressed using normalization and constraints  

- Handling duplicate entries  
  → Managed through validation and database rules  

- Query performance  
  → Improved through optimized database queries
